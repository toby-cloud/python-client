# python-client
Use Toby with any Python application.

## Usage

```bash
make init  # install dependencies
make test  # run tests
make coverage # run tests with coverage report
make clean # cleanup
```

## Dependencies

 - [pytest](http://doc.pytest.org/en/latest/)
 - [paho-mqtt](https://pypi.python.org/pypi/paho-mqtt/1.1)
